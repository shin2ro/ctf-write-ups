## Write-up

なんとなくスコアサーバーでブラウザの開発者ツールを開くと

> Try exec "i_need_extra_challenge()" via your browser's devtools.

とコンソールに出力されていたので `i_need_extra_challenge()` を実行する。  

🌌🪐と2種類の0幅スペースが使われた文字列が与えられる。  
0幅スペースを0と1に置き換えて文字に変換するとフラグになる。
